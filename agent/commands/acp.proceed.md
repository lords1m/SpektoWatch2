# ACP Proceed

Continue the active task from `agent/progress.yaml`.

## Steps

1. Read `AGENT.md`, `agent/progress.yaml`, and the active task file.
2. Inspect only the code and design files needed for that task.
3. Implement the smallest coherent change.
4. Run targeted validation.
5. Update the task file and `agent/progress.yaml` if task state changes.
6. Report changed files and validation results.
