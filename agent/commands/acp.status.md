# ACP Status

Summarize the current repository state for an agent handoff.

## Steps

1. Read `AGENT.md`.
2. Read `agent/progress.yaml`.
3. Read the active milestone and task files.
4. Check `git status --short`.
5. Report:
   - current milestone
   - active task
   - completed task count
   - next action
   - uncommitted files relevant to the current task
